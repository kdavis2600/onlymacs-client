OnlyMacs technical install summary

Mac menu bar app, CLI, and approved local AI routes.

This package installs the current OnlyMacs product surface described by the README standard: one native Mac app, one branded command path, and explicit trust boundaries for file-aware work.

  • Installs OnlyMacs.app, the local bridge, and the onlymacs launcher used by /onlymacs.
  • Optionally installs Codex and Claude Code skills/slash commands so supported tools can call the same launcher.
  • Uses Ollama for local model hosting when available; the app exposes setup and starter-model controls after install.
  • Routes work across Public, Private, and Local-First modes. File-aware remote work requires explicit approval and read-only export; no public repo browsing or hidden write-back.
  • Includes the in-app recipe section for common /onlymacs review, generation, transform, planning, and local-first commands.

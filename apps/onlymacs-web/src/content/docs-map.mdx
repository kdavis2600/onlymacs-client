# Docs Map

Use this map to find the right docs without starting from terminal commands.
Most everyday workflows begin in the menu bar app and continue in Claude Code or
Codex with `/onlymacs`.

<Learn>
Use the app for status and repair. Use Claude Code or Codex for requests. Use
the CLI Reference only for automation, advanced diagnostics, or support-led
troubleshooting.
</Learn>

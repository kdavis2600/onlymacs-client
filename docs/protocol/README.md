# OnlyMacs Protocol Notes

Human-readable notes for the Sprint 1 protocol scaffold.

## Current scope

- shared envelope structure
- auth handshake seed messages
- coordinator-backed provider registration
- swarm creation, invite minting, and join semantics
- bridge-local swarm create/join and invite surfaces for the macOS app
- exact-model visibility and preflight surfaces for the bridge and coordinator
- session reservation/release surfaces for dynamic slot accounting
- bridge runtime mode and active-swarm selection
- local share publish/unpublish with Ollama model discovery
- local Ollama-backed OpenAI-compatible chat relay for the proof-of-concept path

- provider capability snapshots over live sessions
- swarm CRUD messages
